Quickstart

What is mTracker?

mTracker is a tool which facilitates data entry for neuroscience experiments. The goal is to facilitate centralisation of datasources. Data is stored in a relational database. This database can be queried in subsequent analysis.

The mTracker Data Model

mTracker uses a hierarchical datamodel. Each dataset belongs to a mouse. Each mouse belongs to a group. Each group belongs to an experiment. Genotype, surgery and session information are specified at the group level. Datasets contain information from one mouse on one session. The full data model can be viewed here.
